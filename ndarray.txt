ndarray 객체 생성 방법

numpy.array(etc...)

전달인자   설명
object    배열을 반환하는 함수 또는 데이터 목록

dtype     배열요소의 데이터 타입

copy      객체의 복사 여부

order     배열의 메모리 저장 구조 지정. C(row major) or F(column major)

subok     True이면 배열을 상속받은 클래스일 경우 상속 클래스 형식, False 이면
          기본 배열 클래스 형식으로 생성

ndmin     결과 배열의 최소 차원(dimensions)수

arange    내장 range함수와 유사하나 리스트 대신 ndarray 반환

linspace  주어진 데이터 범위를 일정한 간격으로 분할해서 배열을 만든 후 튜플로 반환

ones      주어진 dtype으로 배열을 생성하고 내용을 모두 1로 초기화

zeros     ones와 같지만 내용을 0으로 채움

empty     메모리를 할당하여 새로운 배열을 생성하지만 값을 초기화 하지않음


*numpy 자료형
